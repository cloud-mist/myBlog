判定质数 \( \to \) 试除法

备注

质数定义

 质数是指在大于 1 的自然数中，除了 1 和它本身以外不再有其他因数的自然数。

朴素 ->O(n)

  1. 小于 2 的数都不是质数
  2. 如果 2-n 中的数能把 n 整除，说明不是质数

优化 ->O(sqrt(n))

  1. 因数是成对出现的，若 d|n ,则有 n/d | n.
  2. 所以 \[d \leq \frac{n}{d} \implies d^{2}\leq n \implies d \leq \sqrt{n}\]

func isPrime(n int) bool {
	if n < 2 {
		return false
	}
	for i := 2; i <= n/i; i++ {
		if n%i == 0 {
			return false
		}
	}
	return true
}

func divide(n int) {
	for i := 2; i <= n/i; i++ {
		if n%i == 0 {
			s := 0
			for n%i == 0 {
				n /= i
				s++
			}
			fmt.Printf("%d %d\n", i, s)
		}
	}
	if n > 1 {
		fmt.Printf("%d %d\n", n, 1)
	}
}
